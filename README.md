# Mensa Clínica

Prototipo de sistema de formulación nutricional clínica.

## Cómo correr el proyecto

Abrir terminal en esta carpeta y ejecutar:

```bash
npm install
npm run dev
```

Abre el navegador en http://localhost:5173

## Acceso

- **Cocina** — acceso directo
- **Clínico** — contraseña: `clinico2025`

## Deploy a Vercel

1. Sube esta carpeta a un repositorio de GitHub
2. Ve a vercel.com y conecta tu cuenta de GitHub
3. Importa el repositorio
4. Vercel detecta Vite automáticamente
5. Click en "Deploy"
6. Obtienes un link público que funciona en cualquier dispositivo
